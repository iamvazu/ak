01 — Current State Audit

What exists today — and what's holding it back

The current avinashkotekar.com is built on WordPress with Elementor. It functions, but it doesn't perform — not in speed, not in storytelling, and not in the way it positions Avinash's brand against the caliber of clients he serves (Marriott, Red Bull, JW Marriott, Westin).

CRITICAL

WordPress + Elementor Bloat

Heavy page weight, render-blocking scripts, dozens of unused CSS/JS files. Page load exceeds 5s on 3G. No image optimization pipeline — raw JPEGs served at full resolution.

CRITICAL

Generic Portfolio Layout

Work is presented in a flat grid with no project storytelling, no case studies, no context. Visitors see thumbnails but don't understand the craft, process, or brand impact.

CRITICAL

Placeholder Testimonials

Fake testimonials with lorem ipsum text from "Alex Johnson, Scrum Master" and "Max Jackson, UI/UX Designer" — not real clients. This destroys credibility instantly.

UX ISSUE

Confusing Navigation

"Work" and "Portfolio" are separate pages with the same content. Redundant nav links, duplicate menus, and no clear user journey from landing to inquiry.

UX ISSUE

No Mobile Optimization

Hamburger menu overlaps content. Images don't lazy-load. Touch targets are too small. No consideration for mobile-first browsing (60%+ of traffic).

BRAND

Weak Brand Positioning

No clear value proposition above the fold. The site opens with a logo and slider but doesn't communicate: who, what, for whom, or why choose Avinash over competitors.

"The current site is a brochure. The redesign needs to be an experience — one that makes the visitor feel the quality of Avinash's work before they even see a single photograph."

02 — Reference Analysis & Creative Direction

Learning from YK Produce and Awwwards winners

What YK Produce does brilliantly

Cinematic opening with numbered scene counter and full-bleed imagery that transitions with rhythm. The menu is an experience itself — full-screen overlay with bilingual labels and curved arrow icons. Video integration feels native, not bolted on. The "Playful Minds" marquee adds kinetic energy. The showreel modal with custom controls shows extreme attention to detail. The footer has its own video background. Every pixel is intentional.

What Awwwards photography sites share

Horizontal scroll galleries, parallax depth on images, custom cursors that react to content, smooth page transitions (barba.js / swup), project pages that tell a story with text-image choreography, scroll-triggered reveals that create rhythm, dark themes that let photographs breathe, typography as a design element (not just text), and loading sequences that build anticipation.

02

Visual Design System & Identity

Build a cohesive design language that feels like the luxury brands Avinash works with — dark, cinematic, and editorially refined.

◆
Dark Cinematic Theme

Deep blacks (#0a0a0a) with warm cream text (#f0ece4) and gold accents (#c8a86e). Photographs become the light source — the UI stays in shadow. Film grain texture overlay across the entire site.

Aa
Typography Pairing

Display: Editorial New or Cormorant Garamond (serif, elegant). Body: Satoshi or General Sans (geometric, clean). Mono: JetBrains Mono for labels and metadata. This creates luxury-editorial tension.

▣
Layout Philosophy

Asymmetric grids. Overlapping elements. Generous whitespace (200px+ section padding). Text layered over images with blend modes. Broken grid compositions that feel editorial, not chaotic.

◎
Custom Cursor

Default cursor replaced with a small dot + outer ring. On project hovers, it morphs into "View Project" text circle. On videos, it becomes a play icon. On draggable areas, it shows arrows.

⬡
Image Treatment

Photographs are always edge-to-edge or with intentional asymmetric margins. Hover states add a subtle brightness shift + scale. Images load with a clip-path wipe animation revealing them directionally.

≡
Fullscreen Menu

Menu icon triggers a fullscreen overlay with dark background. Links animate in with staggered vertical slides. A preview image follows the cursor, showing a featured image from each section on hover.

03

Motion Design & Interaction Layer

This is the layer that separates a good portfolio from an Awwwards contender. Every scroll, hover, and page transition should feel choreographed.

"Motion should feel like the website is breathing — alive but not frantic. Like watching a perfectly timed camera pan."

▶
Page Load Sequence

A loading counter (00→100) on a black screen, then a camera-shutter clip-path animation reveals the hero image. Logo and nav fade in after. Total sequence: 2.5 seconds. This builds anticipation.

↕
Scroll-Triggered Reveals

Text blocks reveal with translateY(40px) → 0 + opacity fade. Images reveal with clip-path wipes (left-to-right or bottom-to-top). Staggered delays create rhythm — each element 0.1s after the previous.

↔
Horizontal Scroll Gallery

Featured projects section on the homepage uses GSAP ScrollTrigger to convert vertical scroll into horizontal movement. Each project card has parallax depth — images move slower than text.

⇄
Page Transitions

Barba.js or Swup for seamless page transitions. A dark overlay slides up, content fades out, new page content fades in. The hero image of the next project could scale up from the card position (FLIP animation).

◉
Magnetic Buttons

CTA buttons have a magnetic pull effect — when the cursor approaches, the button slightly moves toward the cursor. On hover, the background fills with a wipe animation. Satisfying micro-interaction.

∞
Infinite Marquee

A CSS-only infinite scrolling marquee strip with category names (FOOD · PRODUCT · ARCHITECTURE · LIFESTYLE · JEWELLERY) in large serif type. Pauses on hover. Separates sections with kinetic energy.

04

Performance & Technical Architecture

An award-winning site that loads slowly is just a pretty failure. Performance is not a feature — it's the foundation.

Recommended Tech Stack
Next.js 15 (App Router) React 19 GSAP + ScrollTrigger Barba.js / Swup Lenis (smooth scroll) Framer Motion Tailwind CSS Sanity CMS Vercel (hosting) Cloudinary (images) Sharp (image processing) Three.js (optional effects)
Performance Targets
Lighthouse Performance95+
Largest Contentful Paint (LCP)< 1.5s
Cumulative Layout Shift (CLS)< 0.05
First Input Delay (FID)< 50ms
Total Page Weight (initial)< 800KB
⚡
Image Pipeline

All images served via Cloudinary or Next.js Image with automatic WebP/AVIF conversion, responsive srcset, blur-up placeholders (LQIP), and lazy loading. Hero images preloaded.

📦
Code Splitting

GSAP and animation libraries loaded only on pages that need them. Dynamic imports for heavy components. Intersection Observer triggers animation code only when sections enter viewport.

🔍
SEO Architecture

Server-rendered pages with Next.js for perfect SEO. Structured data (JSON-LD) for photographer schema. Dynamic OG images per project. Sitemap auto-generation. Alt text on every image.

🎬
Video Optimization

Self-hosted MP4 with poster frames. Videos compressed to H.265/VP9 at 720p max for web. Intersection Observer to play/pause based on visibility. No YouTube embeds — custom player only.

05 — What Makes This Unique

The award-winning differentiators

📷
Camera Shutter Loading

The initial page load mimics a camera shutter — black blades close, a counter ticks, then the blades open to reveal the hero. This is the signature moment. Nobody else has this.

🖱️
Cursor as Viewfinder

The custom cursor is styled as a camera viewfinder crosshair. On image hovers, it adds focus brackets. On drag areas, it shows film-strip arrows. The cursor IS the brand.

🎞️
Film Strip Project Navigation

At the bottom of each case study, the "next project" section shows a horizontal film-strip with perforations and frame numbers, scrolling to reveal the next project's images.

🌑
Darkroom Mode Toggle

A subtle toggle that shifts the entire site to red safelight tones — like being in a darkroom. An easter egg for photography enthusiasts. Unique, memorable, shareable.

📐
EXIF Data Reveals

On project pages, hovering over a photograph reveals its EXIF metadata (camera, lens, aperture, ISO) in a subtle overlay. This speaks to photo professionals and signals technical mastery.

🎵
Optional Ambient Audio

A sound toggle in the corner. When on, a subtle ambient track plays (like restaurant ambiance for food shoots, nature sounds for lifestyle). Opt-in only. Immersive, never intrusive.

06 — Execution Timeline

From concept to launch

Week 1–2
Discovery & Content Strategy

Audit all existing photography assets. Select 8-10 hero projects for case studies. Write project narratives and brand copy. Gather real client testimonials. Define the visual identity system (colors, fonts, spacing scale).

Week 2–3
UI/UX Design in Figma

Design all 5 page templates in Figma. Create a component library. Prototype key interactions (menu, horizontal scroll, page transitions). Mobile-first responsive designs. Design review and iteration cycle.

Week 3–5
Core Development

Set up Next.js project with Tailwind and Sanity CMS. Build the component library. Implement page layouts, responsive grid, navigation, and routing. Set up Cloudinary image pipeline and build the portfolio data model.

Week 5–7
Animation & Interaction Layer

Integrate GSAP + ScrollTrigger. Build the camera-shutter loading sequence. Implement horizontal scroll gallery, page transitions (Barba.js), custom cursor, scroll reveals, magnetic buttons, marquee strip, and all micro-interactions.

Week 7–8
Performance & Polish

Lighthouse audit and optimization pass. Image compression pipeline finalization. Code splitting and lazy loading. Cross-browser testing (Chrome, Safari, Firefox, Edge). Mobile device testing. Accessibility audit (WCAG 2.1 AA).

Week 8–9
Content Population & QA

Upload all project images and content to Sanity CMS. Final copy editing pass. SEO metadata and OG images. Complete QA across devices. Soft launch for feedback. Final adjustments.

Week 9–10
Launch & Submission

DNS migration and production deployment on Vercel. 301 redirects from old WordPress URLs. Google Analytics 4 + Search Console setup. Submit to Awwwards, CSS Design Awards, FWA. Social media launch campaign.
